# team-5-collaboration-assignment
<h1>Our take on Asteroids</h1>

- Destroy or avoid enemies to gain points
- Getting hit will lose points

--- 

Controls:
- Use the arrow keys or WASD to move around
- Left click to shoot

---

| Action            | Points |
|:-----------------:|:------:|
| Shooting an enemy | +20    |
| Dodging an enemy  | +5     |
| Getting hit       | -50    |
